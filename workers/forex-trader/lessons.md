# 外匯員工教訓

## 策略教訓
- 門檻太高 = 系統癱瘓，防虧防到不做等於白費
- 下單要謹慎、賺的抱住、虧的快砍
- 重點是最終有沒有賺，不是勝率
- 獲利的單子可以繼續抱，不急著平
- 追蹤停損比固定停利好
- **策略退休決定必須立即執行** — W15 教訓：W14 決定 retire 2個策略但未封鎖，多虧 $12.36
- **US30 是毒藥** — 2週0%WR，累計虧 $177+。spread 太高，波動太大，永久黑名單
- **13:00 UTC 不交易** — 美股開盤前波動極大，兩週虧 $410+。等到 14:00 再開始
- **高信心 ≠ 高勝率** — confidence ≥0.75 連續兩週全敗。評分系統根本錯誤
- **量少也是優勢** — W15 量減79%，虧損也減78%。不要為了交易而交易
- **新策略要小量測試** — session_momentum 直接0%WR虧$15。新策略用最小倉+限制筆數

## 風控教訓
- **Circuit breaker 不是可選項** — W15 出現13連敗。3連敗暫停30分鐘回測可節省 $17+/週
- **虧損不要死扛** — 本週虧損交易平均持有776分鐘。120分鐘減倉, 240分鐘全平
- **RR比 1.47 不夠** — 9.5% WR 配 1.47 RR 一定虧。需要 RR≥3.0

## 技術教訓
- Capital.com API 的 epic 代碼要用對（GOLD, EURUSDM2026 等）
- SL/TP 被 broker 觸發時，要主動回查持倉狀態更新 P/L — CLOSED trades 不能 pnl=0
- 不要在 BB 擠壓時開倉
- strategy_registry.json 的 retired 策略必須在引擎中硬性封鎖，不能只改狀態
