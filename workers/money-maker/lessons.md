# 賺錢機員工教訓

## 核心教訓
- RSI>50 不買是死規則，上漲行情永遠進不了場（v3 血淚）
- 趨勢中回調也要給分，RSI 40-75 都有機會
- 門檻太高（4分）= 空轉，降到 3 分後才正常
- 過濾器不能歸零封殺，要用乘數降分
- 冷卻期 30 分鐘太長，15 分鐘夠了
- 追蹤停損分階段：+0.4%保本 → +0.8%追蹤0.5% → +1.5%追蹤0.35% → +2.5%追蹤0.25%
- 只會抄底不會跟趨勢 = 錯過大部分行情

## API 教訓
- MAX API 參數用 snake_case：`ord_type` 不是 `ordType`
- 市價買單需要 `volume`（BTC 數量），不是台幣金額
- BTC/TWD 最小交易量：0.0001 BTC
- BTC/TWD 最小賣出金額：250 TWD
