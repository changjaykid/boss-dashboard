# Carousel Structure Templates

## 🎯 選結構原則
- 同一天兩篇必須用不同結構
- 連續 3 篇不可重複同一結構
- 優先選跟上一篇「感覺不一樣」的結構

---

## 教學型 (Tutorial)
Best for: frameworks, how-to, step-by-step

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 鉤子標題 — 「你可能一直做錯的 X 件事」 |
| 2 | problem | 受眾常見痛點 |
| 3 | steps | 3 個可執行步驟 |
| 4 | detail | 補充說明或注意事項（可選） |
| 5 | cta | 行動呼籲 |

## 故事共鳴型 (Story)
Best for: case studies, behind-scenes, lessons learned, 創業者真實故事

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 故事鉤子 — 「上週遇到一件事…」「他 28 歲負債 200 萬…」 |
| 2 | problem | 情境 / 困境描述 |
| 3 | solution | 怎麼解決 / 觀點轉折 |
| 4 | summary | 教訓 / 洞察 |
| 5 | cta | 引導互動 |

## 觀點輸出型 (Opinion)
Best for: hot takes, myth-busting, contrarian views

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 挑釁標題 — 「很多人搞錯了…」 |
| 2 | problem | 常見迷思 / 錯誤認知 |
| 3 | solution | 你的觀點 + 論據 |
| 4 | cta | 留言討論引導 |

## 工具推薦型 (Tool)
Best for: tool reviews, comparisons

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 工具名稱 + 一句話價值 |
| 2 | steps | 使用場景 / 功能亮點（3 點） |
| 3 | detail | 實測心得 / 優缺點 |
| 4 | cta | 試用引導 |

## 數據洞察型 (Data)
Best for: stats, trends, reports

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 數字 + 標題 — 「73% 的品牌都忽略了這件事」 |
| 2 | problem | 數據背景 |
| 3 | solution | 數據解讀 + 行動建議 |
| 4 | cta | 分享引導 |

## 🆕 對比型 (Comparison / Before-After)
Best for: 工具對決、改前改後、A vs B

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 「X vs Y，誰贏？」或「改之前 vs 改之後」 |
| 2 | problem | A 的特點 / 改前的狀態 |
| 3 | solution | B 的特點 / 改後的狀態 |
| 4 | detail | 直接對比表格或結論 |
| 5 | cta | 你選哪邊？留言告訴我 |

## 🆕 清單型 (Listicle)
Best for: 推薦清單、排行、懶人包

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 「2026 必備的 5 個 XX」「我最常用的 X 個工具」 |
| 2 | steps | 第 1-2 項（每項一句話 + 亮點） |
| 3 | steps | 第 3-4 項 |
| 4 | steps | 第 5 項 + Bonus |
| 5 | cta | 你最推哪個？ |

## 🆕 時間軸型 (Timeline)
Best for: 品牌發展史、產業演變、趨勢變化

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 「從 0 到 1 億的路上，他做對了 3 件事」 |
| 2 | problem | 階段一：起點 |
| 3 | solution | 階段二：轉折 |
| 4 | detail | 階段三：爆發 |
| 5 | summary | 關鍵洞察 |
| 6 | cta | 你現在在哪個階段？ |

## 🆕 問答型 (Q&A / FAQ)
Best for: 常見問題、粉絲提問、知識科普

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | 「你一定問過的 X 個問題」「粉絲最常問我的事」 |
| 2 | problem | Q1 + 簡答 |
| 3 | solution | Q2 + 簡答 |
| 4 | detail | Q3 + 簡答 |
| 5 | cta | 還有什麼問題？留言問我 |

## 🆕 迷思破解型 (Myth Buster)
Best for: 翻轉認知、打臉常識

| Slide | Type | Content |
|-------|------|---------|
| 1 | cover | ❌ 大字寫出常見迷思 |
| 2 | problem | 為什麼大家這樣想（看似合理的原因） |
| 3 | solution | ✅ 真相是什麼 + 數據/案例佐證 |
| 4 | detail | 具體怎麼改（可選） |
| 5 | cta | 你之前也這樣想嗎？ |

---

## Slide Count Guidelines
- **3-4 slides**: Quick opinion, single insight, 迷思破解
- **5 slides**: Standard depth (most types)
- **6-7 slides**: Deep dive tutorial, timeline, detailed comparison
- Agent decides based on content complexity; avoid filler slides
