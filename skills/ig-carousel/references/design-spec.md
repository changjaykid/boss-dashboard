# IG Carousel Design Specification

## Image Specs
- Size: 1080 x 1350 px (4:5 portrait)
- Format: PNG
- Slides per post: 5-7

## 整體設計方向（2026-04-07 全面升級）

**定位**：現代、成熟、有質感、知識型，偏高級雜誌封面與品牌海報。

**絕對不要**：
- ❌ 廉價模板感、電商促銷感、學生簡報感、一般 AI 海報感
- ❌ 卡通感、科技藍炫光、過度炫技特效
- ❌ 太花、太亮、太黑、每張長得一模一樣
- ❌ 置中滿版、太多小圖示、太多框線、資訊塞滿

**要做到**：
- ✅ 像高質感商業內容創作者在經營的版面
- ✅ 大標題主導 ＋ 清楚層級 ＋ 大量留白 ＋ 左對齊排版
- ✅ 一眼先看到標題 → 第二眼看到重點字 → 第三眼才讀補充內容
- ✅ 整個帳號看起來像同一個品牌做的

---

## 排版原則

- **對齊方式**：左對齊為主（非置中）
- **標題**：大、醒目、有份量，是畫面主角
- **內文**：乾淨清楚、易讀，是配角
- **留白**：大量留白，讓畫面有呼吸感
- **層級**：標題 → 重點字 → 內文，三層清楚分明
- **資訊密度**：寧可少放，不要塞滿

## 背景規範

### 絕對不要
- ❌ 純黑或太黑的底
- ❌ 整張壓到很暗、悶住
- ❌ 每天同一張背景

### 背景風格方向
帶有空氣感、光影感、質感的深色或中深色背景：
- 暖灰、深咖、煙燻棕、霧黑、灰褐色
- 低飽和城市夜景、柔和室內空間、窗框光影
- 現代街景、商場走廊、建築空間、玻璃倒影
- 樓梯長廊、都會人物背影或側身

### 背景處理方式
- **柔和壓低亮度**（不是壓到全黑）
- **保留空間層次**（前中後景分明）
- **保留環境細節**（看得出是什麼場景）
- **帶一點景深或霧感**
- 讓文字清楚但畫面仍有呼吸感

### AI 背景生成 Prompt 指引
生成背景時的 prompt 要包含：
- 場景描述（具體的地點/空間類型）
- 色調指定：warm muted tones, soft ambient lighting
- 品質關鍵字：editorial photography, medium format, shallow depth of field, atmospheric haze
- 明確排除：NOT dark, NOT moody, NOT black background, NOT neon
- 亮度：medium-dark exposure, visible details, breathable atmosphere

### bg_darken 參數
- 不再用固定 0.65
- 根據背景亮度動態調整：0.40-0.55
- 原則：能看清文字，但背景細節仍可辨識

## 字體規範

### 使用字體
- 主字體：PingFang TC（macOS）/ Noto Sans CJK TC（fallback）
- 標題：Bold / Semibold，偏粗、有張力
- 內文：Regular，乾淨俐落

### 絕對不用
- ❌ 可愛字體、手寫字體、花體字、過度裝飾字

### 字級參考
- 封面主標：68-76px Bold
- 封面副標：32-36px Regular
- 內頁標題：48-52px Bold
- 內頁內文：34-38px Regular
- UI 元素：22-28px

## 色彩系統

### 文字顏色
- 主文字：米白 / 暖白 / 淺灰白（#F5F0EB / #FFFDF8 / #E8E4DF）
- 不要純白 #FFFFFF（太刺眼）

### 品牌強調色（5 選 1，固定一段時間使用）

| 方向 | 色碼參考 | 適合場景 |
|------|---------|---------|
| 焦糖橘 / 琥珀橘 | #D4874D / #C07A3E | 溫暖、專業、商業感 |
| 奶油金 / 暗金米色 | #C9A96E / #B8975A | 高端、品牌、質感 |
| 霧紅棕 / 陶土色 | #A0695A / #8B5E50 | 文藝、深度、故事感 |
| 鼠尾草綠灰 / 橄欖灰綠 | #7A8B6F / #6B7D5E | 清新、自然、有機感 |
| 冷灰藍 / 霧藍灰 | #7B8FA1 / #6A7F91 | 理性、科技、冷靜 |

**目前使用**：焦糖橘 #D4874D（暫定，可更換）

### 色彩規則
- 一張圖最多只用一個強調色 + 白字系統
- 不要螢光色、不要太俗豔
- 整體克制、高級、耐看

---

## 品牌一致性

雖然每天背景場景不同，但以下必須固定：
- 色調方向（暖色系深色調）
- 字體氣質（現代、乾淨、有力）
- 排版邏輯（左對齊、大標主導、三層層級）
- 留白感（不塞滿）
- 強調色（同一段時間用同一個）

## Fixed UI Elements（每張必有）

| Position | Element |
|----------|---------|
| Top-left (60, 40) | @freeeadman (26px, 暖白半透明) |
| Bottom-left | SHARE (20px) |
| Bottom-right | SAVE (20px) |
| Bottom-center | Progress dots + page number (e.g. 1/5) |
| Right-center | Slide arrow › (非最後一張) |

UI 元素要低調，不搶畫面。

## Slide Types

### cover — 封面（最重要）
- 大標題是主角（68-76px Bold）
- 可選副標（32-36px）
- 大量留白
- 必須一眼抓住注意力

### content — 內容頁
- 左對齊小標（48-52px Bold，強調色）
- 內文（34-38px Regular，暖白）
- 段落間充足留白
- 重點字可用強調色

### steps — 步驟頁
- 編號用強調色（大字）
- 每步驟一行或兩行
- 不要擠

### highlight — 重點金句
- 一句話放大，居中或左對齊
- 字級 52-60px
- 留白更多

### cta — 行動呼籲
- CTA 標題（52-56px Bold）
- 引導文（34px）
- 可選：底部細線 button 框（不要真的做按鈕感）
