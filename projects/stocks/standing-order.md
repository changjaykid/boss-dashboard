# 股票專案 (Stock Project) - Standing Order

## 🎯 任務目標
每天整理 **10 檔可能有上漲機會的股票**。
- **條件限制**：其中至少 **5 檔股價低於 50 元**。
- **分析範圍**：前五個工作天的所有台股資料。

## 📊 輸出內容 (每檔股票必備)
1. **名稱**
2. **代號**
3. **產業類別**
4. **公司介紹**
5. **為什麼推薦**
6. **該注意什麼事**

## 🔄 每日復盤與自我優化要求
每次產出新報告前，**必須先針對「前一次」的推薦名單進行復盤**：
1. **追蹤記錄**：記錄哪些判斷是對的，哪些是錯的。
2. **自我優化建議**：
   - 哪些選股邏輯不夠準？
   - 哪些產業判斷需要補強？
   - 哪些資料來源可以增加？
3. **優化落實**：完成自我優化檢討後，必須自行記錄並將新邏輯應用於當日的選股中。

## ⚠️ 嚴格限制
- **不可捏造資料**：所有的價量與基本面必須基於真實市況。
- **資料不足需標示**：若有任何資訊無法取得或不完整，必須明確標示「待補」。
- **理由必須充分**：絕對不可只給出股票名稱而不附帶明確推薦理由。
