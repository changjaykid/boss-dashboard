## 22:43 重大教訓：git push 覆蓋 JSON 資料

**問題**：git add -A + push 把已更新的 social-data.json 蓋回舊版，導致已發佈的文章又跑回待審核
**根因**：沒有先 git pull，且用 git add -A 把所有檔案都加進去
**老闆非常不滿**，這種低級錯誤不能再犯

### 永久修正規則
1. push 前必須 `git pull --rebase`
2. 只 `git add <specific files>`，絕不用 `git add -A`
3. push 完檢查 JSON 資料正確性
4. 改 HTML 和改 JSON 分開 commit，不混在一起
